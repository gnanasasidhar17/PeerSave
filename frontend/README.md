## üöÄ **Getting Started**

### **Prerequisites**
- Node.js 16+ 
- npm or yarn

### **Installation**

1. **Install dependencies:**
   ```bash
   cd frontend
   npm install
   ```

2. **Start development server:**
   ```bash
   npm start
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

### **Environment Variables**
Create a `.env` file in the frontend directory:
```env
REACT_APP_API_URL=http://localhost:5000/api
```

## üîß **Customization**

### **Themes**
The app uses a dark theme by default. You can customize colors in `tailwind.config.js`:

```javascript
colors: {
  primary: {
    // Your custom primary colors
  },
  accent: {
    // Your custom accent colors
  }
}
```

### **Components**
All components are modular and reusable. Customize them in the `src/components` directory.

### **Styling**
Uses Tailwind CSS with custom utilities. Add new styles in `src/index.css`.
